# Cloudwawi — Mockups

Static, clickable UI prototypes for the Cloudwawi procurement automation app.
Each file is self-contained HTML (inline CSS/JS); data is mocked in-browser and
actions don't persist.

## Live site

Published via GitHub Pages: **https://sparkdynamic-gmbh.github.io/mockups/**
